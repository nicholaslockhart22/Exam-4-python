# K-mer Context Counter

This project counts k-mers from DNA sequence fragments and records the frequency of the character that immediately follows each k-mer.

## Files

- `kmer_context.py`: Main Python script.
- `test_kmer_context.py`: Pytest test file.
- `example_sequences.txt`: Example input file.

## How to run the script

```bash
python kmer_context.py example_sequences.txt 2 output.txt
